# tampermonkey

Para utilizar os Snippets, precisa ter instalado em seu Browser, a extensão [TamperMonkey](https://chrome.google.com/webstore/detail/tampermonkey-beta/gcalenpjmijncebpfijmoaglllgpjagf)
